# Medimap
It's a Health-Care Web App built in MERN Stack that provides immediate set of Doctors available for Emergency health issues and more.
In emegency situations people generally find it difficult and time consuming to find doctors one by one in different hospitals, most importantly when time is crucial for the patient.
This problem is solved by Medimap. Medimap provides doctors with specialization in the specific health issues, available at that time and in the locations selected by the user.
